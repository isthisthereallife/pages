Play along whilst watching a film!

Features
  • best domain name = tropes.bingo
  • Look up work (i.e. a film, a book)
  • Generate a bingo board
  • Watch/Read/Think about said thing
  • Tick the box when you feel that what is portrayed in said box happens in the thing you're watching/reading/thinking about
  • Shout *BINGO* when you win

Written in Javascript, and some java for the backend
